# 安装Cursor

进入官网：https://www.cursor.com/

点击下载即可：

![image-20241204162136193](https://cdn.jsdelivr.net/gh/letengzz/tc2/img202412041621148.png)

![image-20241204162520775](https://cdn.jsdelivr.net/gh/letengzz/tc2/img202412050002411.png)

注册登录此时会显示该账号的信息，每个账号的模型调用次数是有限的，其中GPT4和Claude3.5的免费调用次数为500次，其它比较弱的模型的调用次数无上限(包括新推出的o1-mini)：

![image-20241204162650483](https://cdn.jsdelivr.net/gh/letengzz/tc2/img202412050002382.png)

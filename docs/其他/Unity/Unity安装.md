# Unity 安装

## 安装 Unity Hub

[Unity Hub](https://unity.com/cn/download) 是使用Unity必装的软件，这个软件可以帮助管理Unity软件的版本和Unity项目。

### 方式一

在浏览器中输入网址 [unity.com/cn/download](https://unity.com/cn/download) 进入下载页面。

点击 下载：

![image-20241015142048122](https://cdn.jsdelivr.net/gh/letengzz/tc2/img202410151424450.png)

### 方式二

在浏览器中输入 [unity.cn](https://unity.cn/releases) 进入Unity官方网站。

![image-20241015142412439](https://cdn.jsdelivr.net/gh/letengzz/tc2/img202410151452419.png)

## 账号注册

进入注册页面后，按照提示进行注册。
填入 **用户信息**；
勾选 **服务条款和隐私政策**
点击 **安全验证**，验证通过后会显示 **重新验证（已验证）**
最后点击 **创建Unity ID**
![image-20241015142940748](https://cdn.jsdelivr.net/gh/letengzz/tc2/img202410151429428.png)

稍等片刻后会出现激活邮件的页面。
![账号注册页面 2](https://i-blog.csdnimg.cn/blog_migrate/cb0af765f9a9c7a218fe93463f3ac003.png)

按照提示，打开刚刚注册时使用的邮箱，在收件箱里找到这封邮件。然后点击 **Link to comfirm email**。
![image-20241015143020647](https://cdn.jsdelivr.net/gh/letengzz/tc2/img202410151430970.png)

出现类似提示页面时，点击 **继续访问**：
![image-20241015143037850](https://cdn.jsdelivr.net/gh/letengzz/tc2/img202410151430438.png)

然后点击 **安全验证**，完成操作后点击 **验证**：
![image-20241015143057857](https://cdn.jsdelivr.net/gh/letengzz/tc2/img202410151430252.png)

## 账号登陆

如果已经有Unity账户，并且已在移动设备上安装了Connect App，可以 **点击以显示二维码** 扫码登陆。

也可以点击 **账户登陆**，使用手机或邮箱登陆。

如果没有Unity账户，可以点击 **创建Unity ID**。

![image-20241015142646450](https://cdn.jsdelivr.net/gh/letengzz/tc2/img202410151501697.png)

点击 **账户登陆** 后，选择登录方式输入登录后即可：

![image-20241015142829136](https://cdn.jsdelivr.net/gh/letengzz/tc2/img202410151501934.png)

## 安装Unity Hub

双击打开**Unity Hub 安装包**。

点击 **我同意**：

![image-20241015143521914](https://cdn.jsdelivr.net/gh/letengzz/tc2/img202410151435066.png)

点击 **浏览** 选择安装路径，建议安装到除C盘以外的其他盘，然后点击 **安装**：

![image-20241015143701152](https://cdn.jsdelivr.net/gh/letengzz/tc2/img202410151437760.png)

安装过程结束，若已勾选 **运行 Unity Hub**，则点击 **完成** 后会自动运行 Unity Hub：

![image-20241015143741428](https://cdn.jsdelivr.net/gh/letengzz/tc2/img202410151501343.png)

## 使用Unity Hub

运行 Unity Hub 后，如果已有 Unity 账号，则点击 **Sign in** 进行登陆。如果没有 Unity 账号，则点击 **Create account** 进行注册。

![image-20241015143839118](https://cdn.jsdelivr.net/gh/letengzz/tc2/img202410151438964.png)

登陆成功后，会弹出安装 Unity 编辑器的提示框。
**例如** 下图中 **2022.3.50f1 LTS** 为官方推荐最新的长期支持版，可以点击 **浏览** 选择安装路径，然后点击 **Install Unity Editor** 开始安装。
也可以点击 **Skip installation** 跳过此安装步骤。

![image-20241015143948517](https://cdn.jsdelivr.net/gh/letengzz/tc2/img202410151440075.png)

获取个人许可证，直接点击 **Agree and get personal edition license** ，即可开始使用 Unity Hub：

![image-20241015144206141](https://cdn.jsdelivr.net/gh/letengzz/tc2/img202410151442743.png)

## 选择安装 Unity 编辑器

在浏览器中输入 [unity.cn](https://unity.cn/) 进入Unity官方网站：

![image-20241015144339761](https://cdn.jsdelivr.net/gh/letengzz/tc2/img202410151443391.png)

长期支持版适用于希望长期持续开发和发布游戏/内容，并期望长时间保持稳定版本的用户。
对于想体验最新且稳定功能的新手来说，推荐使用长期支持版。
点击 **长期支持版**，选择最新的那个下载：

![image-20241015144459399](https://cdn.jsdelivr.net/gh/letengzz/tc2/img202410151445362.png)

推荐使用 从Unity Hub 下载，若浏览器弹出此提示，勾选 **始终允许 unity.cn 在关联的应用中打开此类链接**，然后点击 **打开**

![image-20241015144611049](https://cdn.jsdelivr.net/gh/letengzz/tc2/img202410151446882.png)

稍等片刻后，Unity Hub 会弹出如下图的安装提示框。
Unity Hub 会默认勾选安装推荐的代码编译器，如 **Visual Studio Community **，这个在写代码时会用到。
如果你已安装编译器，那么可以取消勾选，安装好后在Unity中重新设置默认的编译器即可。
然后点击 **Continue** 继续。

![image-20241015144728618](https://cdn.jsdelivr.net/gh/letengzz/tc2/img202410151501062.png)

若弹出此提示框，**勾选** 后点击 **Install** 开始安装：

![image-20241015144744304](https://cdn.jsdelivr.net/gh/letengzz/tc2/img202410151501569.png)

若弹出此提示框，点击 **是** 即可：

![image-20241015150055879](https://cdn.jsdelivr.net/gh/letengzz/tc2/img202410151500601.png)

当出现： Installation Failed时，关掉Unity Hub并使用管理员重新运行并重试。
